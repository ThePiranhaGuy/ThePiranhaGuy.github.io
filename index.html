2023-02-03

Predicting sanity test results | Convolve Hackathon

"For the first time ever, 6 IITs along with Cisco have come together to organize Convolve, an ML/AI Hackathon"
and I got in on it and won Third Prize. Here's the approach:

• Processed a dataset of sanity test results on Github commits with categories pass, fail, and aborted
• Transformed data using data imputation and expanding the combined tests into individually performed tests
• Employed clustering,feature crossing and decomposing making features having higher dependency to target
• Tested various target encoding methods and applied RandomForest achieving 96.5% accuracy in classification.
• Created a pipeline with 2-step classifier to deal with class imbalance yeilding accuracy score of 98.601%

2022-10-27

Gender-Bias Detection in Text | Open IIT Data Analytics 2022-23

An annually held Data Analytics Hackathon at IITKGP that aims to test out ability to approach a problem uniquely. Damn good PS they have.
I was main man behind the keyboard: testing, coding and running models. Our efforts landed us the Third place. Kinda sad tbh.
Here's the approach:

• Classified individual sentences using a depth-first search approach into known biased and unbiased labels
• Performed text preprocessing/cleaning while keeping pronouns intact in order to identify three kinds of bias
• Employed text vectorizing techniques and used an ensemble of SVM, XGBoost, and CATboost to predict
• Achieved a sentence pair accuracy of 98.89 using a second ensemble layer of Voting, BERT, and Bi-LSTM
